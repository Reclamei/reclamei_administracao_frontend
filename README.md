## Instalação & Execução

Tenha o angular instalado em escopo global, se não tiver, o faça com o seguinte comando:

```npm install -g @angular/cli```

Instale as dependências do projeto com o comando:

```npm install```

Execute o projeto com o comando:

```npm start```

ou

```ng serve```
